## 🔎 Repository types

| Type | Meaning | Typical output |
|---|---|---|
| `package` | Reusable software capacity | functions, documentation, tests |
| `analysis` | Reproducible technical workflow | tables, figures, models, diagnostics |
| `report` | Operational or institutional document | PDF, DOCX, HTML report |
| `dashboard` | Interactive visualization or app | Shiny, Quarto dashboard, web app |
| `manuscript` | Scientific publication workflow | paper, supplement, figures, tables |
| `institutional` | Organization support repository | templates, handbook, workflows, environment |
